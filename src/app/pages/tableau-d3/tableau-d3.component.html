Tableau → D3 Concept Map

Every Tableau concept has a D3 equivalent. This page maps them 1-to-1 so you can reason about your migration.

Shelves & Cards → Scales

In Tableau, you drag fields onto shelves. In D3, scales map a data domain to a visual range.

TableauD3 EquivalentNotes
Columns shelf (categorical)scaleBand()Evenly spaced bands with padding
Columns shelf (continuous)scaleTime()Date/time x-axis
Rows shelf (measure)scaleLinear()Numeric domain to pixel range
Rows shelf (log scale)scaleLog()Logarithmic scale
Color shelf (categorical)scaleOrdinal(schemeTableau10)D3 ships Tableau's own palette
Color shelf (continuous)scaleSequential(interpolateBlues)Value to color gradient
Size shelfscaleSqrt()sqrt for area-based sizing (bubbles)

Marks → SVG Elements

Tableau's Mark type sets what shape represents each row. In D3, you choose the SVG element directly.

Tableau MarkD3 / SVG ElementNotes
Bar<rect>x, y, width, height from scales
Line<path> via d3.line()Line generator computes d attribute
Area<path> via d3.area()Fills between two y values
Circle<circle>cx, cy, r from scales
Pie / Donut<path> via d3.arc()d3.pie() computes angles, d3.arc() draws them
Text label<text>Append to SVG or arc centroid
Map / Polygon<path> via d3.geoPath()GeoJSON → SVG path
