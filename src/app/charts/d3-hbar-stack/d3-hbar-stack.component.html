Data Model

Stacking — d3.stack()

d3.stack() transforms flat rows into layered series with cumulative [start, end] pairs per segment.

Rendering Stacked Rects — layers + inline labels

Each stack layer becomes a <g> with a shared fill. Labels are rendered inside segments only when the width exceeds a threshold.

Tooltip — attachTooltip() helper

The attachTooltip helper shows all three segment values with color indicators on hover.

Key D3 Concepts Used
