Horizontal Stacked Bar Chart

NPS-style comparison using d3.stack() with inline labels and score column

◆
Tableau Equivalent
Drag a dimension to Rows, multiple measures to Columns, set Analysis → Stack Marks → On. Mark type = Bar.

Data Model

Stacking — d3.stack()

d3.stack() transforms flat rows into layered series with cumulative [start, end] pairs per segment.

Rendering Stacked Rects — layers + inline labels

Each stack layer becomes a <g> with a shared fill. Labels are rendered inside segments only when the width exceeds a threshold.

Tooltip — attachTooltip() helper

The attachTooltip helper shows all three segment values with color indicators on hover.

Key D3 Concepts Used

d3.stack()
Transforms flat data into cumulative [start, end] layers
scaleLinear(0–100)
Percentage x-axis with fixed domain
Inline labels
Centered text inside segments, visibility gated by width
NPS score column
Computed promoters − detractors shown before each bar
