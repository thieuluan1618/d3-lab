Data Model — multi-period

Wrap each period's rows in a PeriodData object. The chart renders one panel per period (small multiples).

Shared Scales — across periods

The y-axis is shared so categories align. The x-scale domain is computed from the max across all periods for consistent bar lengths.

Small Multiples — one panel per period

Loop over periods, translate each panel by i * (panelWidth + gap). Each panel gets its own x-axis and period title.

Value Labels — text at bar ends

Append <text> elements positioned at x(d.value) + 4 and vertically centered with dominant-baseline: middle.

Legend Highlight — data-key selector

Uses data-key attribute selectors instead of CSS classes to handle names with spaces correctly.

Tooltip — attachTooltip() helper

The attachTooltip helper wires mouse events. Tooltip title includes the period label for context. Use <ng-content> to project custom Angular components.
