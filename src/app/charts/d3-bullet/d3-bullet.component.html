Data Model

Background Tracks

Full-width gray rect elements drawn first so value bars render on top.

Rendering Bars & Labels

Value bars use scaleLinear for width. Labels are positioned at the end of each bar.

Tooltip — attachTooltip() helper

The attachTooltip helper wires mouse events + opacity toggle on the value bars.

Reference Line

A dashed vertical line at x = 100 serves as the index baseline.

Key D3 Concepts Used

Background tracks
Full-width gray rects behind value bars for visual context
Reference line
Dashed vertical line at a constant value (index baseline)
Sorted data
Data sorted descending so highest values appear at top
Value labels
Text elements positioned at bar ends for direct readout
