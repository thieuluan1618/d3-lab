Data Model

Line Generator — d3.line() with configurable curves

d3.line() generates an SVG path. The .curve() method accepts any D3 curve factory — toggle between monotoneX, linear, step, basis, or cardinal using the buttons above the chart.

Data Point Dots

Grid Lines — the extended tick trick

Create a second axis with tickSize(-width) and no labels — gives horizontal grid lines.

Tooltip — multi-series aggregation

Hover a dot to see all series values at that month. Unlike single-series charts that use attachTooltip, line charts wire mouse events manually to aggregate across series.

Legend — horizontal with highlight

Click a legend item to highlight that series (others dim to 12% opacity). Click again to reset. Uses drawLegend with layout: 'horizontal' and onHighlight callback.

Responsive Resize — observeResize()

All charts use a shared observeResize helper that wraps ResizeObserver with debounce. On resize: clear SVG, re-read container width, and re-draw at the new size.

Key D3 Concepts Used

d3.line()
Path generator: array → SVG path string
.curve(factory)
Configurable interpolation: monotoneX, linear, step, basis, cardinal
d3.scalePoint()
Ordinal scale with fixed spacing (no bands)
datum() vs data()
datum binds entire array to one path element
observeResize()
ResizeObserver + debounce for responsive re-render
drawLegend(onHighlight)
Click to highlight series, click again to reset
