API Contract — flat response from BE

Segments are top-level and own their own flat data[] of market × year × column records. columns[] is a deduped lookup shared across all segments. Finalized shape for the backend team.

UI Data Model — nested for rendering

Transform — flat → nested

ChartTransformService.heatmapResponseToChart() groups the flat rows into segments → geographies → years → cells.

Color Scale — D3 sequential scale

scaleSequential maps a continuous domain to a color interpolator like interpolateOranges.

Rendering — nested HTML tables

One <table> per segment, with rowspan for market groups and [ngStyle] for cell colors.

Key D3 Concepts Used

d3.scaleSequential()
Maps a continuous domain to a color interpolator
d3.interpolateOranges
Light → dark orange color ramp
rowspan
HTML attribute to merge market group cells
[ngStyle]
Angular binding for dynamic inline styles
