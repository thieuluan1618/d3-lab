Data Model

Color Scale — D3 sequential scale

scaleSequential maps a continuous domain to a color interpolator like interpolateYlOrRd.

Rendering — HTML table with ngStyle

No SVG needed — use a standard <table> and bind D3-computed colors via [ngStyle].

Key D3 Concepts Used

d3.scaleSequential()
Maps a continuous domain to a color interpolator
d3.interpolateYlOrRd
Yellow → Orange → Red color ramp
d3.min() / d3.max()
Auto-detect domain from data values
[ngStyle]
Angular binding for dynamic inline styles
