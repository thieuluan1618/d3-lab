Data Model

Layout — d3.pie() + d3.arc()

d3.pie() computes angles. d3.arc() converts angles to SVG paths. Set innerRadius > 0 for a donut chart.

Rendering Slices

Tooltip — attachTooltip() helper

The attachTooltip helper wires mouse events to each slice. Access d.data for the original PieDatum. Use <ng-content> to inject custom Angular components.
