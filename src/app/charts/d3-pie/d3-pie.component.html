Pie Chart

Pie chart using d3.pie() layout + d3.arc() path generator

◆
Tableau Equivalent
Mark type = Pie. Drag a dimension to Color, a measure to Angle/Size.

Data Model

Layout — d3.pie() + d3.arc()

d3.pie() computes angles. d3.arc() converts angles to SVG paths. Set innerRadius > 0 for a donut chart.

Rendering Slices

Tooltip — attachTooltip() helper

The attachTooltip helper wires mouse events to each slice. Access d.data for the original PieDatum. Use <ng-content> to inject custom Angular components.

Key D3 Concepts Used

d3.pie()
Layout that computes startAngle/endAngle
d3.arc()
Path generator from arc angles
arc.centroid()
Returns [x,y] midpoint of an arc
scaleOrdinal(schemeTableau10)
Tableau's own 10-color palette
