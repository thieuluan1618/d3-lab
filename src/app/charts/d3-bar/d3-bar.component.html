Data Model

Scales — the key to every bar chart

scaleBand divides the x-axis into evenly spaced bands. scaleLinear maps values to height.

Rendering Bars — the join pattern

D3's .join() binds data to SVG <rect> elements — one rect per data point.

Tooltip — attachTooltip() helper

The attachTooltip helper wires mouse events + opacity toggle. Use <ng-content> to project custom Angular components into the tooltip.

Key D3 Concepts Used

d3.scaleBand()
Maps categories to pixel bands with padding
d3.scaleLinear()
Maps numeric domain to pixel range
